- Assignment operators assign a value to a variable
- | Operator | Example | Same as |
  | = | `x = 5` | `x = 5` |
  | += | `x += 3` | `x = x + 3` |
  | -= | `x -= 3` | `x = x - 3` |
  | *= | `x *= 3` | `x = x * 3` |
  | /= | `x /= 3` | `x = x / 3` |
  | %= | `x %= 3` | `x = x % 3` |
  | **= | `x **= 3` | `x = x ** 3` |
